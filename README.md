# ProjetoReact

Projeto iniciado com a proposta de automatizar o atendimento de encomendas artesanais do cliente, melhorando o processo de coleta de informações da encomenda até o processo de recebimento de atualizações pela parte do cliente

# Tecnologia

O projeto está sendo desenvolvido usando React, Node.JS e Postgree Sql para o armazenamento dos dados.
